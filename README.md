# ZhongWén Diario 中文日记

App web interactiva para aprender **chino mandarín** desde cero, dirigida a hispanohablantes y basada en la estructura del libro *Chino Contemporáneo para Principiantes* (当代中文).

## Características

- **21 lecciones** (Lección 0–20) con 30+ caracteres/palabras cada una
- **4 apéndices**: índice de caracteres, trazos, radicales y respuestas
- Tarjetas 3D volteables con pinyin, tonos, radicales y ejemplos
- Cuadrícula de escritura **米字格** con calco digital
- **10 minijuegos**: memorama, dictado, rueda de tonos, conversación, etc.
- **6 tipos de ejercicios** del libro digitalizados
- Sesión diaria estructurada (calentamiento → vocabulario → escritura → juego → cierre)
- Sistema de **rachas**, metas diarias, logros e **repaso espaciado (SM-2)**
- Audio TTS en mandarín (Web Speech API) con modo shadowing
- Modo claro/oscuro · Diseño responsive

## Inicio rápido

```bash
npm install
npm run dev
```

Abre `http://localhost:5173` en el navegador.

## Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm run preview` | Previsualizar build |

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS 4 + Framer Motion
- React Router · localStorage para progreso
